# 基于模板匹配的单数字简易OCR识别系统
## 项目简介
本项目使用MATLAB实现传统图像处理单数字识别，依靠灰度化、中值滤波、Otsu二值化、形态学运算、连通域筛选、相关系数模板匹配完成0~9数字识别，轻量化无需GPU。

## 运行环境
- MATLAB R2022b+
- Image Processing Toolbox 图像处理工具箱

## 文件结构
- Direction3.m 主程序入口
- template/ 0-9标准数字模板库
- test_images/ 测试样本图片
- 机器视觉大作业-平台 完整实验报告

## 使用步骤
1. MATLAB打开项目文件夹
2. 运行Direction3.m
3. 弹窗选择待识别数字图像
4. 自动输出各阶段处理图像、匹配置信度与识别结果

## 算法流程
图像载入→灰度化→中值滤波→Otsu二值化→自动反色→形态学开闭运算→连通域裁剪→32×32归一化→模板相关系数匹配

## 实验效果
印刷数字识别准确率100%，单图推理耗时＜0.05s

## 改进方向
角度校正、多字体模板库、局部阈值分割、嵌入式C++移植
